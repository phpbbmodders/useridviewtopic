# User ID in Viewtopic

Adds the user id in viewtopic.

## Installation

1. Copy the extension to: `/ext/phpbbmodders/useridviewtopic`
2. In the Administration Control Panel, navigate to: **Customise → Manage extensions**
3. Enable the **User ID in Viewtopic** extension
